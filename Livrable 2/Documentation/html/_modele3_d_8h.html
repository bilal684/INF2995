class  modele::Modele3D
 Classe qui encapsule un modèle 3d de la librairie 'assimp'. Cette classe permet de charger un modèle 3d d'un fichier exporté par un outil (en utilisant ladite librairie). La classe ne permet pas, dans sa forme actuelle, de modifier le modèle par la suite.
