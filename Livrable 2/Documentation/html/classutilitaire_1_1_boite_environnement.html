Detailed Description

Classe représentant une boîte d'environnement ("skybox").

Elle s'occupe de charger 6 images du cube formant la boîte. Elle utilise la convention de sens de CubeMapGen (de ATI), lorsque les images sont exportées avec le mapping OpenGL (plutôt que DirectX).
