Detailed Description

Classe qui implémentante le visiteur concrêt permettant de parcourir l'arbre de rendu et de visiter chaque noeud afin d'annuler une rotation.

Les visiteurs concrêts pourront faire différents traîtements en fonction du type de noeud visité, car ils possedent des attributs differents.
