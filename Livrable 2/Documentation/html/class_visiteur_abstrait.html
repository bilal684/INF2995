Detailed Description

Classe de base du patron visiteur qui permet de rajouter des des fonctionnalités aux noeuds présents dans l'arbre de rendu sans avoir à modifier la hiérarchie elle-même. Donc on parcourt modifie les éléments présents dans la l'arbre de rendu sans avoir à modifier leurs classes de base. Il convient d'implémenter différents visiteurs concrêts qui surchargeront la méthode visiter() autant de fois qu'il y a d'objets à manipuler.

Les visiteurs concrêts pourront faire différents traîtements en fonction du type de noeud visité.
