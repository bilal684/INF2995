Cette fonction convertit une chaîne de caractères en un type générique. Comme elle se sert de l'opérateur >> appliqué sur un istringstream, elle devrait fonctionner pour toutes les classes qui définissent cet opérateur pour les istream&.

Parameters
[in]chaine: Chaîne à convertir.
[in]valeur: Valeur template qui contiendra la valeur contenu dans la chaîne. Ne sera pas modifiée si la chaîne n'est pas valide.
[in]modifieur: Fonction utilisée pour la conversion de type particulier, par exemple std::hex ou std::oct pour les entiers.
Returns
Vrai si la conversion a réussi.
