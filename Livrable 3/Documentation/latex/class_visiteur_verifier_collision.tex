\subsection{Detailed Description}
Classe qui implémentante le visiteur concrêt permettant de de vérifier la collision entre le robot et les objets poteaux et murs. 

Les visiteurs concrêts pourront faire différents traîtements en fonction du type de noeud visité, car ils possedent des attributs differents.
