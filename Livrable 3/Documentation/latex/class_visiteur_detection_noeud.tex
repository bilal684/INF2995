\subsection{Detailed Description}
Classe qui implémentante le visiteur concrêt permettant de parcourir l\textquotesingle{}arbre de rendu et de visiter chaque noeud afin de determiner s\textquotesingle{}il y a une collision ou non avec les capteurs de distances. 

Les visiteurs concrêts pourront faire différents traîtements en fonction du type de noeud visité, car ils possedent des attributs differents.
